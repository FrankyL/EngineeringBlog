12 Buttons ESP32-based Macropad with Hot-swap sockets (V2)

Franky Laurentis November 24, 2024

Based on the ESP32 and a multiplexer.

3D printed casing, ESP32 can send combination of keystrokes to PC through BLE, V2 can store up to 2 sets of 11 stroke combinations.

V1 has basic functions

V2 has LEDs for each buttons

Upcoming V3 have OLED to display icons of what each button does, improved upon V2 that shows with LED colour which set is currently active

In the V1 and V2, Wi-Fi SSID, passwords, color of RGB LEDs, and keystrokes combinations also all stored within EEPROM, and I honestly think it is way too troublesome to be an actual serious product to sell to the wider public.

V3 should also have an SD card with config file to make this easier, also more suitable for storing large number of combinations of keystrokes + icons for OLED screen.
