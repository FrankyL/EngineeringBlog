12 Buttons ESP32-based Macropad with Hot-swap sockets (V2.5)

Franky Laurentis January 3, 2025

An improvement over the earlier V2 Macropad and solved many of its issues.

Solved issues:

  1. More than 2 sets of 11 combinations. Originally the V2 can only have 22 combinations since it is very hard to remember which buttons, of which set, does what. Now the OLED just shows it.

  2. OLED icons display which set is currently active and the icons for the actions, in V2, can only use the color to know which set is active.

  3. SD card can now store a large number of combinations and is much more intuitive than the V2's approach of using SERIAL UART.

  4. SD card stores a table of set index, combination index, button combinations, the English Icons and the Chinese Icons.

  5. Can store separate icons within a folder. Currently icon is .bmp with 32x32 size, really like how it looked like as 16x16 but it is too small in actual OLEDs so I just scaled it up.

Currently, new issue is the laggy OLED display updates, took around 500-700 ms to change. Perhaps the issue is due to using 3 separate 128x32 OLEDs and the resulting workaround of using software I2C instead of the hardware. Will make the actual V3 with 128x96 or 128x128 OLEDs to see the difference, this issue is strange since I clearly remember 128x64 is actually fast enough to do animation.
